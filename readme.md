### Part 1: Analysis
需求：实现一个可以完成更多功能的计算器。

1. 输入表达式时可以得出结果，如 $2+3$，$5+2*3$等
2. 支持括号描述优先级，如$(5+2)*3$
3. 支持定义变量，如$x=3$
4. 支持数学函数，如$sqrt(3)$
5. 支持高精度，如$9999999999999999999.222222222+1.11$
